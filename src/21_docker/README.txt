# README

Das Install-Script ist die Version OHNE "small" - das ist für das Einbinden in der Doku.
Das Script muss evtl zuerst mit 

       chmod +x docker_install.sh

ausführbar gemacht werden. 
Der Start erfolgt dann als 

       sudo ./docker_install.sh
       
Für die Ausführung als "BENUTZER":

    usermod -aG docker BENUTZER
